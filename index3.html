Pembayaran melalui Alfamart

1. Datang di gerai Alfamart terdekat

2. Info kasir untuk pembayaran DOKU ICONNet

3. Info kasir NO Pembayaran, Contoh : 111113000000001

4. Pembayaran : sesuai dengan jumlah pemesanan

5. Transaksi Selesai

6. Simpan Struk Pembayaran

Pembayaran melalui Indomaret

1. Datang ke gerai Indomaret terdekat

2. Info kasir untuk pembayaran Stroomnet

3. Info kasir kode indomaret 88267001 diikuti 8 digit Kode Pembayaran Contoh : 8826700110021136

4. Pembayaran sesuai dengan jumlah tagihan ditambah dengan biaya Rp 6.450,-

5. Transaksi Selesai

6. Simpan Struk Pembayarann

Pembayaran melalui mandiri atm

1. Masukkan kartu ATM dan PIN

2. Pilih menu BAYAR/BELI

3. Pilih menu LAINNYA -> LAINNYA -> INTTV

4. Masukkan kode Perusahaan / Institusi , yaitu “89450” (ICONNET)

5. Masukkan NO VA, Contoh : 894500000001

6. KONFIRMASI : Tekan 1 kemudian tekan OK

7. Transaksi Selesai

8. Simpan Struk Pembayaran

Pembayaran melalui mandiri online

1. Buka aplikasi mandiri online di handphone atau PC

2. Masukkan user id dan pin di halaman log in

3. Pilih menu BAYAR -> BUAT PEMBAYARAN BARU

5. Pilih menu INTERNET -> ICONNET/89450

6. Pilih Rekening Sumber

7. Masukkan NO VA

8. Tekan Lanjut

9. Tekan KONFIRMASI

10. Masukkan MPIN Banking kemudian tekan OK

11. Transaksi Selesai

Pembayaran melalui mandiri cabang

1. Datang ke Cabang Bank Mandiri

2. Isi slip setoran atau pindah buku, Langkah Pengisian Aplikasi Setoran / Transfer

3. Isi Tanggal, Nama & Alamat Pembayar

4. Isi penerima dengan kode 89450

5. Isi No.Rekening dengan NO VA

7. Isi Jumlah Setoran & Terbilang

8. Isi Tujuan Transaksi. Misal: PEMBAYARAN XXXX

9. ntarkan slip setoran ke teller

10. Teller akan memproses & mencetak bukti pembayaran

Pembayaran melalui MCM / MIB

1. Buka aplikasi MCM / MIB

2. Masukkan company id, user id dan password

3. Pilih menu PEMBAYARAN -> PEMBAYARAN TAGIHAN

4. Pilih rekening debet

5. Pilih KATEGORI INSTITUSI (89450/ICONNET)

7. Masukkan NO VA

8. Tekan LANJUT -> KONFIRMASI

9. Approve transaksi dengan memasukkan pin dinamis token

10. Transaksi selesai

Pembayaran melalui ATM BRI

1. Masukkan kartu ATM dan PIN

2. Pilih menu Transaksi Lain -> LAINNYA -> BRIVA

3. Masukkan Nomor Virtual Account , yaitu “1326891” (ICONNET)

4. Masukkan NO VA, Contoh : 1326891000000001

5. Di halaman KONFIRMASI, pastikan detil pembayaran sudah sesuai Nomor BRIVA, Nama Pelanggan dan Jumlah Pembayaran

6. KONFIRMASI : Tekan 1 kemudian tekan OK

7. Transaksi Selesai

8. Simpan Struk Pembayaran

Pembayaran melalui Mobile Banking BRI

1. Buka aplikasi BRI Mobile

2. Pilih menu Mobile Banking BRI -> PEMBAYARAN -> BRIVA

3. Masukkan Nomor Virtual Account

5. Masukkan Jumlah Pembayaran

6. Masukkan PIN

7. Transaksi SelesaiSimpan notifikasi SMS sebagai bukti pembayaran

Pembayaran melalui Internet Banking BRI

1. Login pada alamat Internet Banking BRI (https://ib.bri.co.id/ib-bri/Login.html)

2. Pilih menu Pembayaran Tagihan > Pembayaran > BRIVA

3. Masukkan Nomor Virtual Account

4. Di halaman konfirmasi, pastikan detil pembayaran sudah sesuai seperti Nomor BRIVA, Nama Pelanggan dan Jumlah Pembayaran

5. Masukkan password dan mToken

7. Transaksi Selesai

8. Simpan Struk Pembayaran

Pembayaran melalui BCA Mobile

1. Pilih menu m-BCA

2. Masukkan kode akses

3. Pilih menu m-transfer

4. Pilih BCA Virtual Account

5. Masukkan Nomor Virtual Account

6. Masukkan PIN m-BCA

7. Transaksi Selesai

8. Simpan Struk Pembayaran

Pembayaran melalui BSB Mobile

1. Login

2. Pilih tab transaksi dan menu Virtual Account

3. Pilih rekening tabungan yang akan didebet

4. Masukkan nomor VA/ID Pelanggan (disertai kode provider)

5. Pastikan tagihan dan keterangan sesuai

6. Masukkan otp dan pin

7. Transaksi selesai
